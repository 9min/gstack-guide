# /gstack-autoplan

> 한 번의 명령으로 CEO, 디자인, 엔지니어링 리뷰를 순차 실행하여 완성된 기획안을 만들어주는 자동 리뷰 파이프라인.

## 무엇을 하는 스킬인가

autoplan은 gstack의 세 가지 리뷰 스킬(CEO 리뷰, 디자인 리뷰, 엔지니어링 리뷰)을 하나의 파이프라인으로 묶어 자동 실행합니다. 각 리뷰 스킬의 전체 내용을 읽어들인 뒤, 6가지 의사결정 원칙에 따라 자동으로 판단하면서 순차적으로 진행합니다.

리뷰 과정에서 판단이 애매한 지점 — 비슷한 접근 방식 사이의 선택, 스코프 경계선의 결정, codex와 의견이 갈리는 부분 — 은 "맛(taste) 결정"으로 분류하여 마지막에 한꺼번에 사용자에게 보여줍니다. 자동으로 처리할 수 있는 부분은 자동으로, 사람의 감각이 필요한 부분만 모아서 물어보는 구조입니다.

결과적으로 하나의 명령만으로 기획안이 세 가지 관점에서 검증되고, 사용자는 최종 판단만 내리면 됩니다.

## 언제 쓰나

- 새로운 기능의 기획안을 작성한 뒤 한 번에 전체 리뷰를 돌리고 싶을 때
- CEO, 디자인, 엔지니어링 리뷰를 각각 따로 실행하기 번거로울 때
- PR을 올리기 전에 기획 완성도를 빠르게 검증하고 싶을 때
- "auto review" 또는 "run all reviews"라고 말할 때
- 리뷰 과정의 반복적인 판단을 자동화하고, 중요한 결정만 직접 하고 싶을 때

## 어떻게 시작하나

`/gstack-autoplan`을 입력하면 CEO 리뷰부터 시작합니다. CEO 리뷰가 완료되면 디자인 리뷰, 이어서 엔지니어링 리뷰가 자동으로 실행됩니다. 각 단계에서 자동 판단이 이루어지며, 맛 결정이 필요한 항목들은 마지막에 정리되어 표시됩니다.

## 실제 사용 예시

새로운 대시보드 기능의 기획안을 작성했습니다. `/gstack-autoplan`을 실행하니 CEO 리뷰에서 스코프 확장 제안 2건, 디자인 리뷰에서 타이포그래피 개선 3건, 엔지니어링 리뷰에서 아키텍처 이슈 1건이 발견되었습니다. 자동 판단 가능한 항목은 바로 반영되었고, "대시보드에 실시간 알림을 추가할지"와 "차트 라이브러리 선택" 두 가지 맛 결정만 사용자에게 물어봤습니다. 10분 만에 세 관점의 리뷰가 완료된 기획안이 나왔습니다.

## 음성 입력

AquaVoice, Whisper 등 음성 입력 도구에서 다음 표현으로 실행할 수 있습니다:

- "auto plan"
- "automatic review"
